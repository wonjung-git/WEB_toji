개인정보 처리방침

본 서비스는 조회 기능 제공을 위해 다음 정보를 처리할 수 있습니다. 도로명 주소, VWorld API 키. API 키는 사용자의 브라우저 로컬 저장소에만 저장되며 서버로 전송하거나 별도로 보관하지 않습니다.

수집한 정보는 조회 요청 처리 목적 외로 사용하지 않으며, 관련 법령에 따라 필요한 경우를 제외하고 제3자에게 제공하지 않습니다.

사용자는 브라우저의 로컬 저장소 삭제를 통해 언제든지 저장된 API 키를 삭제할 수 있습니다.
